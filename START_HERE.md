## 🎮 Try These First

### Add More Products
Edit `src/utils/products.js`:
```javascript
{
  id: 9,
  name: "Rose Garden",
  category: "Women",
  price: 79.99,
  // ... other properties
}
```

### Change Colors
Edit `src/styles/index.css`:
```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
}
```

### Add a New Page
1. Create `src/pages/NewPage.jsx`
2. Create `src/styles/NewPage.css`
3. Add route in `src/App.jsx`
4. Add link in Header
